Overview: This application provides integrations for M-Pesa with various monitoring and documentation tools.

  • Swagger Documentation: Access the API documentation at /swagger-ui.html.
  • Actuator Endpoints: View application health and metrics at /actuator.
  • Grafana Dashboard: Monitor performance with Grafana at /grafana.
  • Prometheus Dashboard: Track metrics and performance data at /prometheus.

Key Endpoints:

  • /api/mpesa/auth: Obtain an authentication token.
  • /api/mpesa/c2b: Initiate a Customer to Business (C2B) transaction.
  • /api/mpesa/b2c: Make a Business to Customer (B2C) payment.
  • /api/mpesa/transaction-status: Check transaction status.
